│                                          1.1 Product Vision                                          │
│                                                                                                      │
│ Create a comprehensive social platform exclusively for university students in China that integrates  │
│ career opportunities (gig marketplace), community building (groups), and essential student lifestyle │
│ services (restaurant reviews, house rentals, dating) into one unified ecosystem.                     │

│                                                                                                      │
│ To be the #1 digital platform for university students to connect, collaborate, earn income, find     │
│ housing, discover food, and build meaningful relationships within their campus community.            │
│                                                                                                      │
│                                           1.3 Target Users                                           │
│                                                                                                      │
│  • Primary: University students in Heilongjiang Province, China                                      │
│  • Secondary: University graduates (within 2 years), prospective students                            │
│  • Initial Launch: Students from partner universities with official collaboration                    │
│                                                                                                      │

│                                                                                                      │
│  • Daily Active Users (DAU): 30,000                                                                  │
│  • Monthly Active Users (MAU): 800,000                                                               │

│                                        4.2.4 Order Management                                        │
│                                                                                                      │
│ Priority: P0                                                                                         │
│                                                                                                      │
│ Order Flow:                                                                                          │
│                                                                                                      │
│  1 Buyer places order:                                                                               │
│     • Clicks "Order Now" → Fills requirements                                                        │
│     • Payment (WeChat Pay/Alipay) → Funds held in escrow                                             │
│     • Order created → Seller notified                                                                │
│  2 Seller delivers:                                                                                  │
│     • Uploads deliverables (files/images)                                                            │
│     • Marks as "Delivered" → Buyer notified                                                          │
│  3 Buyer reviews:                                                                                    │
│     • Accepts delivery → Funds released to seller (minus 10% commission)                             │
│     • OR requests revision (1-2 revisions allowed)                                                   │
│     • OR disputes (admin mediation)                                                                  │
│  4 Auto-completion:                                                                                  │
│     • If no action in 7 days → Auto-accepted                                                         │
│                                                                                                      │
│ Order States:                                                                                        │
│                                                                                                      │
│  • Pending (awaiting seller acceptance - 24h timeout)                                                │
│  • In Progress (seller working)                                                                      │
│  • Delivered (waiting buyer review)                                                                  │
│  • Revision Requested (needs changes)                                                                │
│  • Completed (funds released)                                                                        │
│  • Disputed (admin review)                                                                           │
│  • Cancelled (refund processed)                                                                      │
│                                                                                                      │
│ Acceptance Criteria:                                                                                 │
│                                                                                                      │
│  • ✓ Payment success rate >98%                                                                       │
│  • ✓ Escrow system 100% reliable                                                                     │
│  • ✓ Dispute resolution within 3 days                                                                │
│  • ✓ Clear order status tracking                                                                     │
