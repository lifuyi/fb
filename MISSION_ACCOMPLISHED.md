## 🚀 立即开始使用

### 1️⃣ 启动项目（只需2步）
```bash
# 进入前端目录
cd frontend

# 启动开发服务器
npm run dev

# 打开浏览器访问
# http://localhost:3000
```

### 2️⃣ 阅读文档
```
1. 先看: GETTING_STARTED.md
2. 再看: frontend/QUICKSTART.md
3. 深入: frontend/README.md
```

### 3️⃣ 连接后端
```bash
# 编辑环境变量
nano frontend/.env.local

# 修改为你的后端地址
NEXT_PUBLIC_API_URL=http://your-backend-url/api
```

- ✅ **快速加载** - 自动代码分割和优化
- ✅ **App Router** - 最新的路由系统
- ✅ **React生态** - 丰富的组件和工具

### 为什么选择 TypeScript？
- ✅ **类型安全** - 减少运行时错误

- ✅ **重构友好** - 大型项目易维护
